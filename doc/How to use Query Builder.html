How To Use Query Builder:
  • When opening the application you should see the below layout:



  • You can use the "Ticket IDs" Field to put text containing the ticket IDs you need, and then press the "Generate" button. Yo should get a Remedy query with the ticket IDs that you can use in the Advanced search bar in Remedy.
NOTE: As you can see in the below example the application filters out all the un-needed text and generate the query with the ticket IDs only.



  • You can also create queries with setting Filtering criteria by checking and unchecking the checkboxes (the query will only be based in the filtering criteria if the "Ticket IDs" field is empty). This can be done for:
1- Priority
2- Status



3- INC/SR
4- Monitoring/Customer
5- SLA Status




6- Service
7- Assigned Group
8- Company



9- Submit start and/or end date
10- Modified start and/or end date
11- Resolved start and/or end date
